PAGINA ELABORADA PARA LAS CONSTANTES Y SUS FUNCIONES.

1:"HTML": FUNDAMENTO PARA INICIAR NUEVO PROYECTO.

2:"TITLE": CONSTANTE PARA PONER TITULO.

3:"/HEAD,BODY":ESTAS CONSTANTES SE PONEN DESPUES DEL TITULO.

4:"h1,/h1": CONSTANTE PARA PONER SUBTITULOS SE MAXIMO 6: "h6".

5:"BODY BGCOLOR": CONSTANTE PARA PONER COLOR DE FONDO.

5:"IMG SRC": CONSTANTE PARA INSERTAR IMAGEN.

6:"WIDTH": CONSTANTE PARA AJUSTAR ANCHO DE LA IMAGEN.

7:"HEIGHT": CONSTANTE PARA AJUSTAR ALTO DE IMAGEN.

8:"BACKGROUND.": CONSTANTE PARA INSERTAR FONDO CON IMAGEN.

9:"BACKGROUND.-SIZE: COVER": AJUSTA LA IMAGEN PARA QUE CUBRA TODA LA PANTALLA.

10:"MARQUEE": CONSTANTE PARA QUE UNA IMAGEN O GIF SE MUEVA.
